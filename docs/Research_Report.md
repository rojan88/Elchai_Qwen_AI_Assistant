# Research Report

## Project title

Elchai AI Workflow Assistant using Qwen

--

## Selected Model

**Qwen 3 (Open-Source Large Language Model)**

Qwen is an open-source family of Large Language Models developed by Alibaba. It is designed for Natural language understanding, code generation,document analysis, reasoning, and AI agent applications. since, it is open-source, organizations can deploy it locally, reducing dependence on commercial AI services while maintaining greater control over sensitive business data.

---

## Why Qwen?

Qwen was selected because it offers:

- Strong coding capabilities
- Fast inference speed
- Long-context understanding
- Open-source deployment
- Lower operational cost
- Good documentation and community support

These characteristics make it a practical option for organizations looking to experiment with AI without relying entirely on paid APIs.

---

## Practical Project

The project demonstrates a simple **AI Workflow Assistant** for Elchai Group.

---

## Conclusion

Qwen demonstrates strong potential as an internal AI assistant. It can reduce repetitive work, improve productivity, and serve as a foundation for future AI agent development while mainteining human oversight.
